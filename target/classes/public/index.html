Catalog Assignment

  1. serving an HTML page generated by a Jtwig template:
    Jtwig template page
  2. serving a list of the catalog items encoded in JSON (use the GSON library)
    Jtwig template page (same page)
  3. adding a new catalog item (via GET query parameters)
    syntax: /addAlbum/title/artist/genre
    e.g. /addAlbum/Pop/U2/Rock
  4. In the generated HTML page, add Javascript code to:
    1. retrieve the list of items and then use DOM methods to display the data on the page
      Shown by clicking on the Album title in the list rendered by template, will go to details page
    2. (BONUS) send a request to create a new catalog item using input from the user (using text boxes, buttons, etc.)
      Jtwig template page (same page)
